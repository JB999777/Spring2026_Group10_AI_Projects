# CyberSafe Checker

## Description
A simple web app that checks messages and tells users if they are likely phishing, with explanations.
Most people cannot tell if an email is a scam or safe, which leads to security risks.
Non-technical users who want help identifying phishing emails and improving basic cybersecurity habits.

## Features
- Input email/message text
- Select urgency level (low, medium, high)
- Add optional context
- Classifies messages as Likely Safe or Possible Phishing
- Provides a risk score
- Gives a simple explanation
- Highlights red flags found in the message

## Live App
https://phishing-detector--jbaleta.replit.app 

## Tools Used
- Replit
- OpenAI

## How It Works
The app analyzes user input and looks for common phishing patterns such as urgency, requests for sensitive information, and suspicious wording. It then generates a classification, risk score, and explanation in simple terms.
